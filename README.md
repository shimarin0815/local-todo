# 📝 ローカル・ToDoリスト

サーバーを使わず、ブラウザだけでサクサク動く、シンプルで美しいToDoリストアプリです。
「**localStorage**」という仕組みを使っているため、ページを閉じたりリロードしたりしても、あなたのタスクは消えずに残ります。

## ✨ 特徴

  * **🚀 準備いらず**: サーバーやデータベースの設定は不要。ファイルを1つ開くだけで動きます。
  * **💾 自動保存**: `localStorage` を使用。タスクを書いたりチェックを入れたりすると、自動的にブラウザに保存されます。
  * **🎨 モダンなデザイン**: 見やすくてきれいな、洗練されたデザインです。
  * **📱 シンプル**: 難しい機能は一切なし。「追加」「完了」「削除」だけのシンプル設計。

## 📂 フォルダ構成

ファイルはこれ1つだけです！

```text
local-todo/
└── index.html  （アプリ本体）
```

## 💻 使い方

特別なソフトのインストールは必要ありません。

1.  フォルダの中にある **`index.html`** をダブルクリックします。
2.  ブラウザ（Google Chrome, Edge, Safariなど）でアプリが開きます。
3.  **タスクを追加**: 入力欄に文字を入れて `Enter` キーまたは「追加」ボタンを押します。
4.  **完了にする**: 左側の四角をクリックしてチェックを入れます。
5.  **削除する**: タスクにマウスを乗せると出る `✖` ボタンを押します。

## 🛠️ 使われている技術

中学2年生でも学習すれば作れる、Webの標準的な技術だけで作られています。

  * **HTML5**: ページの骨組み
  * **CSS3**: デザインとアニメーション
  * **JavaScript (Vanilla JS)**: アプリの動きとデータ保存（`localStorage`）

## ⚠️ 注意点

  * データは「今使っているブラウザ」の中に保存されます。別のブラウザや別のパソコンで開くと、データは共有されません。
  * ブラウザの履歴やキャッシュ（Cookieなど）を完全に削除すると、タスクも消えることがあります。
